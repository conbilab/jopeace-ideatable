각자의 고민을 테이블 위에 올려놓고, 다른 분야 사람의 질문과 실제 경험으로 다시 보는 시간입니다.

01

강의보다 대화에 가깝습니다

누군가가 일방적으로 알려주는 시간이 아니라, 참가자 각자의 고민과 아이디어를 함께 질문합니다.

02

네트워킹보다 문제 해결에 가깝습니다

명함을 나누고 끝나는 자리가 아니라, 지금 막힌 부분을 다른 관점으로 다시 보는 시간입니다.

03

추상적 조언보다 실제 경험을 나눕니다

개발자, 디자이너, 마케터, 소상공인, 콘텐츠 제작자가 각자의 현실을 바탕으로 말합니다.

말하다 보면 생각이 정리됩니다

혼자 굴리던 생각이 질문을 만나면 실행 가능한 다음 행동으로 더 빨리 선명해집니다.

모임은 이렇게 진행됩니다.

    내 고민을 꺼냅니다

  2. 2단계

    질문을 받습니다

    내가 당연하게 생각한 부분을 다른 사람이 질문해 주며 막힌 지점을 보여줍니다.

  3. 3단계

    경험을 나눕니다

    비슷한 문제를 겪은 사람과 전혀 다른 업종의 사람이 각자의 현실을 말합니다.

  4. 4단계

    아이디어를 구체화합니다

    막연했던 생각을 실행 가능한 다음 행동으로 정리합니다.

  5. 5단계

    연결을 이어갑니다

    필요하면 이후에도 서로의 사업, 콘텐츠, 프로젝트를 지켜보며 도움을 주고받습니다.
